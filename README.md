Eu treinei um pouco e agora você pode me contar tudo num arquivo só! Eu chamei ele de `sorteio.json` que é um tipo de arquivo especial para mim!

As informações são as mesmas! Você pode me dizer usando os campos abaixo, se for texto é melhor que esteja entre `aspas '""'` mas se for número não tem esse problema.

- marcar

- amigos
  - Nesse campo você me diz quantos amigos eu tenho que marcar a cada comentário! É só escrever um número! Igual o nome dos Androids!!

- comentarios
  - E aqui você conta pra mim quantos comentários quer que eu faça! É outro número! Está entendendo porq o Doutor Maki Gero usou esses nomes??

- pagina
  - Aqui você escreve o nome da página onde está o sorteio! Igual você fez na lista de amigos, mas só me diz uma hein! Ainda não tenho KI sufuciente para mais do que isso!

- publicacao
  - Eu ainda não sei achar a publicação do sorteio.. Você pode me dizer onde está? É só contar, a partir da mais recente, o número do android que representa essa publicação!
